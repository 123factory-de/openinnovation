## Overview

AWS Global Startup Program은 AWS 기반으로 구축하는 후기 단계의 제품 준비 완료 스타트업을 위해 설계되었습니다. 코셀 기회와 기술 지원, 시장 진출(go-to-market) 지원을 제공해 스타트업이 AWS 파트너 생태계와 함께 더 빠르게 성장하도록 돕습니다.

## Focus Areas

- Cloud Technology
- Co-Sell
- Go-to-Market
- Technical Enablement

## Collaboration & Benefits

- AWS 영업팀과의 코셀 기회
- 기술 및 아키텍처 지원
- 시장 진출 및 마케팅 지원
- 클라우드 크레딧과 AWS 파트너 네트워크 활용

## How to Apply

- 공식 AWS Global Startup Program 페이지에서 자격 요건을 확인하고 지원하세요.
