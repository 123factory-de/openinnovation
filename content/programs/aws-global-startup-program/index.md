## Overview

The AWS Global Startup Program is designed for later-stage, product-ready startups building on AWS. It offers a curated mix of co-sell opportunities, technical enablement, and go-to-market support to help startups scale faster alongside the AWS partner ecosystem.

## Focus Areas

- Cloud Technology
- Co-Sell
- Go-to-Market
- Technical Enablement

## Collaboration & Benefits

- Co-sell opportunities with AWS sales teams
- Technical and architectural support
- Go-to-market and marketing support
- Cloud credits and access to the AWS partner network

## How to Apply

Review eligibility and apply through the official AWS Global Startup Program page.
