## Overview

The Media Tech Hub Accelerator is run by the Media Tech Hub Potsdam, based in the Potsdam-Babelsberg media region. It supports startups working in media tech, immersive media, and AI, offering mentorship and access to a strong creative and technology ecosystem.

## Focus Areas

- Media Tech
- Immersive Media
- Artificial Intelligence
- Mentorship and Networking

## Benefits

- Mentorship from media tech and industry experts
- Access to the Potsdam-Babelsberg media ecosystem
- Networking with partners, creatives, and technologists
- Support to develop and scale media-focused ventures

- Find program details and application information on the official Media Tech Hub Accelerator page.
