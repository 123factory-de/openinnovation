## 개요

Microsoft for Startups Founders Hub는 초기 B2B 스타트업을 위한 Microsoft의 상시 지원 플랫폼입니다. 소프트웨어와 AI 기반 기업을 대상으로 기술 크레딧과 도구, 멘토링을 제공하며, 파트너 생태계도 활용할 수 있도록 지원합니다.

## 주요 분야

- 소프트웨어
- AI
- 클라우드 통합
- B2B 기술

## 혜택

- 최대 15만 달러 규모의 Azure 크레딧
- OpenAI 모델 이용
- GitHub Enterprise와 Microsoft 365 혜택
- 멘토링과 창업 지원

## 지원 방법

- 공식 Microsoft for Startups Founders Hub 포털에서 직접 신청합니다. 프로그램은 상시 운영되며, 실제 승인 전에는 자격 검증 절차를 거치게 됩니다.
