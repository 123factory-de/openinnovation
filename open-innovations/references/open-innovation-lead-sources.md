# Open Innovation Lead Sources

Last reviewed: 2026-06-09
Purpose: discovery inputs for `collect-open-innovation`, not final evidence

This file lists useful aggregator-style platforms, challenge marketplaces, and open-call hubs that can surface candidate open innovation programs faster than generic search alone.

Use these sources to find leads. Do not use them as the final authority for `status`, `deadline`, `benefits`, `eligibility`, or `applicationPath`. Re-verify all included programs against official sponsor pages, official challenge pages, official application pages, or official press releases.

## How To Use This Reference

1. Start with 2-4 sources that match the user’s scope.
2. Pull candidate program names, sponsors, challenge titles, regions, and URLs.
3. Click through to the official sponsor or application destination.
4. Confirm current status and application path from the official source.
5. If no official source can be verified, move the lead to `Excluded Or Uncertain Leads`.

## Practical Search Patterns

Use the source itself plus targeted search queries.

- `site:f6s.com/challenge "open innovation"`
- `site:herox.com "challenge" startup company`
- `site:innocentive.com/challenges company`
- `site:ninesights.ninesigma.com technology search`
- `site:innoget.com "call for innovation"`
- `site:openinnovationnetwork.gov.sg challenge`
- `site:norcat.org/innovation/open-innovation/challenges`
- `site:circulareconomy.europa.eu/platform/en/funding-opportunities/open-calls startup`

## Selection Heuristics

Prefer leads that have at least one of these traits before spending time verifying them:

- Named sponsor company, agency, or consortium
- Explicit challenge statement or open-call scope
- Clear application destination
- Visible deadline or ongoing-submission signal
- Clear statement that external startups, SMEs, researchers, or solution providers can participate

Deprioritize leads with these patterns:

- No identifiable sponsor
- No outbound official link
- Pure community/profile pages
- Expired cohort pages with no newer cycle
- Media coverage with no application route
- Generic procurement notices with no innovation collaboration component

## Suggested Workflow By User Need

For broad global scans:

- Start with `F6S`, `HeroX`, `InnoCentive`, and `Innoget`
- Add `NineSigma/NineSights` for technical scouting-heavy sectors

For public-sector or challenge-heavy scans:

- Start with `HeroX`, `InnoCentive`, `Open Innovation Network`, and EU open-call hubs

For sector-specific scans:

- Start with a broad platform plus one vertical platform
- Example: mining -> `F6S` + `NORCAT`

For Singapore or Asia-oriented scans:

- Start with `Open Innovation Network`, then broaden with `F6S` and `Innoget`

## Source Notes

- F6S says it is the “Open Innovation Challenge Leader” and highlights corporate open innovation challenges, POCs, accelerators, and internal challenges.
- HeroX describes itself as a global crowdsourcing platform where organizations can post challenges and people can contribute solutions.
- InnoCentive’s challenge page shows active challenges, closing dates, awards, and links to challenge details.
- NineSigma’s official solution-provider guidance says NineSights is its digital innovation platform and repository for projects ranging from technology searches to grand challenges.
- Innoget states that it provides open innovation, technology scouting, technology transfer, and challenge-program capabilities, and it exposes calls for innovation on-platform.
- Wazoku presents InnoCentive as part of its ecosystem and links users to the latest challenges.
- yet2 presents itself as an open innovation and technology scouting intermediary, which makes it useful context but a weaker direct lead directory.
- MIT Solve explicitly says it launches open innovation challenges and also supports custom challenge launches for organizations.
- Open Innovation Network calls itself Singapore’s national gateway to open innovation and exposes ongoing challenges with deadlines and trial opportunities.
- NORCAT describes its platform as an online global marketplace connecting builders and buyers of innovation to identify, test, and deploy mining-sector solutions.
- The European Circular Economy Stakeholder Platform exposes open calls with recipient profiles, deadlines, and official outbound links, which makes it a useful adjacent lead source for EU opportunity discovery.
